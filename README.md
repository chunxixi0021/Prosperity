# Prosperity

AI 期末项目

这是一个使用 Vue.js 和 TypeScript 构建的 Web 应用程序。

## 技术栈

*   **前端:** Vue.js
*   **语言:** TypeScript

## 开始使用

要获取本地副本并运行，请按照以下简单步骤操作。

### 环境准备

确保您已安装 [Node.js](https://nodejs.org/) 和 [npm](https://www.npmjs.com/)。

### 安装

1.  克隆仓库
    ```sh
    git clone https://github.com/chunxixi0021/Prosperity.git
    ```
2.  进入项目目录
    ```sh
    cd Prosperity
    ```
3.  安装 NPM 包
    ```sh
    npm install
    ```

### 运行应用

要启动开发服务器，请运行：
```sh
npm run dev
```
打开浏览器并访问提示的地址 (通常是 `http://localhost:5173`)。
